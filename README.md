# Accumulating Momentum

A productivity web application designed to break through the overwhelemed of starting heavy academic or programming tasks.

## 🚀 Key Features
- **Dynamic Task Initialization:** Converts raw thought or stress text into isolated target concepts.
- **2-Minute Micro-Sprint Timer:** A low-friction countdown clock designed to secure immediate psychological micro-victories.
- **Workspace To-Do Tracker:** A clean interface to manage mission-critical sub-tasks dynamically.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Galaxy Theme Animations)
- Vanilla JavaScript (ES6)

## 💻 Setup Instructions (How to Run)
1. Download or clone this repository to your local machine.
2. Locate the `_study productivity_.html` file in the project folder.
3. Double-click the file to open and run the application instantly in any web browser—no local servers or installation required!
